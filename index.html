Path to Success

For the remainder of the course, I’d like to deepen my understanding of all things Javascript and improve my development time. By the end of the course, I expect to be able to successfully deploy my applications and create a sustainable network with my classmates.

After the course, I plan on sharpening all skills that I’ve learned over the past few months. I do not plan on rushing out to interview in the tech industry. A good place to start would be my GitHub repositories; I would like to go back through every repo created and updated the code until it’s fully functional.

Now that we’re beyond the halfway point, I accept that all skills require practice and I’ll need to pay my dues with the material we’ve covered before I can say that I’ve mastered it to job status strength.
